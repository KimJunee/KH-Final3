Event calendar

JavaScript plugin to display a calendar of events.

Around component

Event calendar component depends on FullCalendar plugin. Make sure to link to FullCalendar css and js files in your document:

  • assets/vendor/fullcalendar/main.min.css
  • assets/vendor/fullcalendar/main.min.js

Use this page as a reference.

You can pass almost any calendar option to data-calendar-options='{}'

Usefull links

  • FullCalendar demo
  • FullCalendar API: options, plugins, methods and events

Premium version

We used open source free version of the plugin for demo purpose. However, you can purchase a premium license to level-up the plugin functionality with premium plugins and support. Check the prices.
